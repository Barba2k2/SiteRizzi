# SiteRizzi

## Publicação no GitHub Pages

Este repositório publica automaticamente no GitHub Pages usando GitHub Actions.

- Workflow: `/home/runner/work/SiteRizzi/SiteRizzi/.github/workflows/deploy-pages.yml`
- Arquivo inicial: `/home/runner/work/SiteRizzi/SiteRizzi/index.html`
- Página principal redireciona para `rizzi-joias-bio.html`

## Como ativar

1. No GitHub, acesse **Settings → Pages**.
2. Em **Build and deployment**, selecione **Source: GitHub Actions**.
3. Faça push na branch `main` (ou rode o workflow manualmente).

Após a execução do workflow, o site ficará disponível na URL de Pages do repositório.
